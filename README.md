# Habit Tracker Client

A modern **React** client application for tracking habits, built with **Vite**, **Tailwind CSS**, **Firebase**, and **Framer Motion**. This project helps users manage, monitor, and maintain their daily routines while providing a clean and interactive UI/UX.

## Features

- User authentication with **Email/Password** and **Google Sign-In** via Firebase.  
- Create, update, and track personal habits.  
- Browse **public habits** and recent habit activities.  
- **Search and filter** habits by category.  
- Set **habit reminders** with time tracking.  
- Responsive design with **Tailwind CSS** and **DaisyUI** components.  
- Interactive UI elements using **Framer Motion**.  
- Animations with **AOS** library and typewriter effects for enhanced UX.  
- Sweet alerts and notifications with **SweetAlert2** and **React Hot Toast**.

## Tech Stack

- **Framework:** React 19  
- **Bundler:** Vite  
- **Styling:** TailwindCSS 4 + DaisyUI  
- **Animations:** Framer Motion, AOS, Typewriter-effect  
- **State Management:** React Context API  
- **Routing:** React Router DOM v7  
- **Backend:** Firebase Authentication  
- **HTTP Requests:** Axios  
- **Notifications:** React Hot Toast & SweetAlert2  
- **Icons:** React Icons
